# POO - Residência em TIC / Software 2026.1 (Serratec)

Este repositório contém os exemplos, exercícios e projetos desenvolvidos durante a disciplina de **Programação Orientada a Objetos (POO)**, referente à Turma 36 de Nova Friburgo.

## 📌 Sobre a Disciplina
A disciplina aborda os pilares da programação orientada a objetos utilizando a linguagem **Java**, focando em boas práticas e padrões de projeto.

* **Instituição:** Serratec / Firjan
* **Localidade:** Nova Friburgo - RJ
* **Semestre:** 2026.1
